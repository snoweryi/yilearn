⚠️ CBAM vs ISO 14064-1 排放範疇定義差異【超高頻考點】
★★★ 必考

CBAM 特殊定義

  • 直接排放（CBAM定義）：
    → 製程燃燒排放
    → 外部輸入熱能/冷能（不管地點，視為直接排放）
  • 間接排放（CBAM定義）：
    → 僅限電力消費產生的排放

ISO 14064-1 定義

  • 類別1直接排放：組織自身燃燒
  • 類別2間接排放：外購電力、蒸汽、熱能、冷能均屬間接
  • ⚠️ CBAM把「輸入熱能」算直接；ISO算間接
⚠️ 官方樣題答案（熱能計算差異）

CBAM排放範疇規定：「與ISO 14064-1不同，外界輸入熱源計入直接排放」→ 正確
鋼鐵/鋁/氫氣：財務調整僅含直接排放（電力間接不計）
水泥/肥料：直接+間接排放均計入財務調整

🔍 豁免與適用地理範圍

✅ 豁免對象（不需遵守CBAM）

  • 冰島、列支敦斯登、挪威（EEA成員）

CBAM計算邏輯與憑證機制
命題方向：憑證數量計算公式、扣抵規則、免費配額調整
📐 CBAM憑證計算基本公式
★★★ 計算必考
CBAM應繳憑證數量（基本公式）
應繳 = 進口產品碳含量 − 原產國已繳碳價折算量 − EU ETS免費配額
每張憑證 = 1公噸CO₂e；憑證價格 = 當週EU ETS配額拍賣週均價
產品碳含量（內含排放量 Embedded Emissions）計算
簡單商品 = 直接排放 + 間接排放（電力）
複雜商品 = 製造過程排放 + 前驅物原料嵌入碳排放
鋼鐵/鋁/氫氣 → 僅直接排放；水泥/肥料/電力 → 直接+間接
EU ETS免費配額調整（2026年加成）
調整後碳含量 = 碳含量 × (1 − 免費配額比例)
2026年加成10%；2027年加成20%；2028年起加成30%
→ 免費配額2026-2034年逐年遞減至0%
📌 季末持有憑證下限規定

每個季度末，進口商CBAM帳戶中持有的憑證數量應不少於當年已進口產品隱含排放量的50%。年底必須達100%。

💰 CBAM抵扣規則（台灣碳費如何抵扣）
①
認可的外顯碳價（有效碳價）
出口國企業已繳納的碳稅費或排放交易費用，可申請抵扣CBAM憑證費用
②
台灣碳費可抵扣 ✅
環境部已確認：台灣碳費屬於「有效外顯碳價」，可依CBAM第9條扣抵，降低企業負擔
③
自願碳市場碳權 ❌ 不可抵扣
自願碳市場購買的碳抵換憑證不被CBAM認可為有效碳價，無法抵扣
📌 CBAM主要目的（考試常出「非」題）
  • ✅ 主要目的：防止碳洩漏（Carbon Leakage）
  • ✅ 確保歐盟境內外企業承擔相當的碳成本（公平競爭）
  • ✅ 促進第三國減碳並推動全球碳定價
  • ❌ 不是主要目的：增加關稅收入、保護主義
📊 CBAM vs EU ETS 連動關係
EU ETS狀況對CBAM的影響
ETS碳價上漲CBAM憑證價格隨之上漲，進口成本增加
ETS免費配額減少歐盟境內廠商成本上升，CBAM維持公平競爭
ETS免費配額歸零（2034年）CBAM全面發揮作用；進口商須100%購買憑證
ETS無拍賣價格當週使用前一週均價計算CBAM憑證價格
📌 2026年憑證價格特殊規定

由於憑證自2027年2月1日才開始販售，2026年進口貨品的CBAM憑證價格將依特定季度之EU ETS配額拍賣平均價格計算（非當週價格）。

🔍 碳含量查驗機構條件（2026年新規）
✅ 查驗機構（Accredited Verifier）資格要求
  • 須符合國際標準：ISO 17029（溫室氣體確查證機構符合性評鑑要求）
  • 由歐盟成員國的國家認可機構（NABs）進行認證
  • 新規：查驗機構可設立於歐盟以外的第三國（如台灣），但仍需由EU認可機構認證
  • 若未經獨立查驗，進口商須改用預設值（Default Values）計算
📌 預設值 vs 實際監測值
  • 優先使用：實際生產監測數據（最準確，成本最低）
  • 若未查驗：使用歐盟公告預設值（通常較高，增加成本）
  • 對台灣企業建議：積極建立碳盤查系統，提供精確監測數據

📌 台灣碳費與CBAM的連結
  • 台灣企業繳納的碳費（環境部）→ 認定為CBAM可抵扣的有效外顯碳價
  • 抵扣依據：CBAM法規第9條「在原產國已支付明訂碳價格」
  • 計算原則：有效支付碳價 = 繳納碳費後扣除任何補貼或回扣的淨額
  • ⚠️ 注意：碳費減免補貼也需扣除才算有效碳價

  • 台灣出口商義務：提供精確直接排放碳含量數據給歐盟進口商（鋼鐵僅計直接排放）
  • 建議：建立ISO 14064-1盤查系統，取得第三方查驗報告
  • 碳費紀錄：保留台灣碳費繳納紀錄，作為CBAM抵扣依據
  • 2026年：歐盟進口商記錄碳含量；2027年9月30日前申報並繳交憑證
情境二：歐盟進口商帳戶第一季末僅持有20噸憑證，但已進口含排放量100噸，是否合規？
  • 規定：季末持有憑證須≥已進口排放量的50%
  • 50%×100噸 = 50噸；實際持有20噸 < 50噸
  • 答：不合規，需立即補購至少30噸憑證
  • 注意：2027年2月1日前憑證尚未販售，2026年才開始記錄
情境三：台灣中小企業出口水泥至歐盟，每年出口量40噸，是否需要遵守CBAM？
  • 水泥屬CBAM管制品，但有小型豁免條款
  • 年進口CBAM產品累積淨重<50噸者免除申報
  • 40噸 < 50噸豁免門檻
  • 答：此情況豁免，無需遵守CBAM申報義務
情境四：判斷哪些碳價可以抵扣CBAM？
  • ✅ 台灣碳費（環境部）→ 可抵扣（有效外顯碳價）
  • ✅ 韓國ETS費用 → 可抵扣
  • ❌ 自願碳市場購買的碳抵換憑證 → 不可抵扣
  • ❌ 台灣碳交所的自願性碳權 → 不可抵扣
  • ❌ 企業自主設定的內部碳定價（影子碳定價）→ 不可抵扣
情境五：CBAM 2026正式期與過渡期最大差異？
  • 過渡期（2023-2025）：每季申報，不需購買憑證
  • 正式期（2026起）：需取得授權申報人資格才能進口
  • 正式期：每年申報前一年全年資料；需購買並繳交CBAM憑證
  • 2026年特殊：申報但憑證2027年2月才販售；2027年9月30日前首次繳交
